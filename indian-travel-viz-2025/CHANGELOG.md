# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-06

### Added
- Initial release of Indian Travel Trends 2025 visualization
- Interactive data visualization with three view modes:
  - International destinations analysis
  - Domestic travel by mode (Rail, Air, Bus)
  - Comprehensive comparison view
- Premium UI design with elegant typography
  - Playfair Display serif font for headers
  - Montserrat sans-serif for body text
  - Gold (#D4AF37) and navy gradient color scheme
- Real-time data from 2025 reports:
  - MakeMyTrip Year-End Report
  - IRCTC/Indian Railways statistics
  - Airport Authority of India data
  - redBus Travel Trends Report
- Responsive design for all devices
- Smooth animations and transitions
- Interactive charts powered by Recharts
- Detailed data source documentation
- GitHub Actions deployment workflow

### Data Highlights
- 950k visitors to Thailand (top international destination)
- 6.9 billion rail passengers in FY24
- 306 million domestic air passengers
- 140 million bus passengers with 25% growth
- 88% growth in pilgrimage travel

### Technical Stack
- React 18.2
- Recharts 2.10
- Vite 5.0
- Modern JavaScript (ES2020+)
